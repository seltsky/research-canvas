한글 사용

한글 노드는 따옴표나 대괄호로 감싸기:
A[가설 1] --> B[검증]

키보드 단축키

  • Cmd/Ctrl + S: 저장
  • Cmd/Ctrl + /: 도움말 토글
  • Esc: 모달 닫기
  • + / =: 캔버스 확대 (편집기 외 영역에서)
  • - / _: 캔버스 축소
  • 0: 100%로 reset
  • F: 화면 넓이에 맞추기
  • Ctrl/Cmd + 마우스휠: 캔버스 zoom

인터랙션

  • Visual 캔버스의 노드를 클릭하면 편집기에서 해당 텍스트가 highlight + 줄로 이동
  • 편집기와 캔버스 사이의 가운데 막대(⋮⋮)를 drag하면 패널 크기 조절
  • 패널 크기는 자동 저장됨

파일 형식

  • .mmd: Mermaid 캔버스 파일
  • .comments.json: 코멘트 thread 파일

AI와 협업하기

  1. 편집기에서 Mermaid 텍스트 전체 복사 (Cmd/Ctrl+A → Cmd/Ctrl+C)
  2. 본인 AI (Claude, ChatGPT 등)에 붙여넣기
  3. "이 부분 이렇게 수정해줘" 요청
  4. AI가 준 새 텍스트를 편집기에 다시 붙여넣기
  5. 우측 캔버스가 즉시 업데이트됨
